# Finance Equations
Some basic financial equations.

## Equations List

1. Loan Payments
2. Future Value
3. Initial Investment
4. Desired Annuity
5. Max Withdrawal
6. Remaining Loan Balance

### Run
1. Clone, cd, install, test.

```
git clone https://github.com/mcolonj/finance.git
cd finance
npm install
npm test
```

### Requirements
Mocha test framework.
```
npm install -g mocha 
```
